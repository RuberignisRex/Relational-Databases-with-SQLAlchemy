# Relational-Databases-with-SQLAlchemy

## Usage Instructions
* Create and activate a virtual environment
* install SQLAlchemy and MySQL Connector
* Run RD_REST_API_Dev.py within this virtual environment

## Expected Result
* The code will initialize an SQL database called shop.db
* add models for Users, Products, and Orders* establish a one-to-many relationship between Users and Orders
* establish a Many-to-Many relationship between Products and Orders
* add data to the database for 2 users, 3 products, and 4 orders
* print the users and their emails in the terminal
* print the products and their prices in the terminal
* print the orders detailing who ordered what
* update the price of product id 2 to 750
* remove user 1 from the database
* and finally close out the session
